## Pattern Assignments

### `app/layout.tsx` (layout, structural-replace)

**Analog:** `app/layout.tsx` (current file, lines 1-33) — keep the shape, swap the content.

**What to preserve from the analog:**
- `import type { Metadata } from "next"` — exact import shape
- `next/font/google` import + variable injection pattern (lines 2-8)
- `export const metadata: Metadata` named export (lines 15-18)
- `export default function RootLayout(...)` with typed children prop (lines 20-33)
- `.variable` applied to `<html>` element (line 28) — DO NOT switch to `.className`

**What to change from the analog:**
- Remove Geist/Geist_Mono imports; replace with single `Quicksand` import
- Change font variable name from `--font-geist-sans` to `--font-quicksand`
- Change `weights` — Quicksand needs `['400', '500', '600', '700']` (4 weights, as string array)
- Change `metadata.title` from `"Create Next App"` to `"Pawnder"`
- Change `metadata.description` to `"Find your perfect furry match"`
- On `<html>`: change `className` from two variables to single `{quicksand.variable}`
- On `<body>`: simplify className — remove `min-h-full flex flex-col` boilerplate

**Structural pattern from analog** (lines 1-33):
```typescript
// SHAPE TO FOLLOW — content swapped
import type { Metadata } from "next";
import { Geist, Geist_Mono } from "next/font/google";   // → single Quicksand import
import "./globals.css";

const geistSans = Geist({                                // → const quicksand = Quicksand({
  variable: "--font-geist-sans",                         // → variable: "--font-quicksand",
  subsets: ["latin"],                                    //   subsets: ["latin"],
});                                                      //   weight: ["400","500","600","700"],
                                                         // })

export const metadata: Metadata = {
  title: "Create Next App",                              // → "Pawnder"
  description: "Generated by create next app",          // → "Find your perfect furry match"
};

export default function RootLayout({
  children,
}: Readonly<{
  children: React.ReactNode;
}>) {
  return (
    <html lang="en" className={`${geistSans.variable} ${geistMono.variable} ...`}>
    //                       → className={quicksand.variable}
      <body ...>{children}</body>
    </html>
  );
}
```

**Why `.variable` not `.className`** (from RESEARCH.md Pattern 5): `.className` applies font directly but does not expose a CSS variable. `.variable` injects `--font-quicksand` as a CSS custom property on `<html>`, which is what `@theme` tokens (`--font-display: var(--font-quicksand)`) resolve against. Without `.variable`, all font-family tokens fall through to `sans-serif`.
